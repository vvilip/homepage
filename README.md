# Minimalistische Homepage

Eine einfache, minimalistische Homepage mit Blog-Funktionalität.

## Struktur

- `index.html` - Hauptseite mit Links und Blog-Übersicht
- `style.css` - Minimalistisches Styling
- `posts/` - Verzeichnis für Blogposts

## Anpassen

1. In `index.html` deine GitHub-, Mastodon- und Email-Links anpassen
2. Neue Blogposts im `posts/` Verzeichnis erstellen
3. Neue Posts zur Übersicht in `index.html` hinzufügen

## Lokale Vorschau

Einfach `index.html` im Browser öffnen oder mit einem lokalen Server:

```bash
python -m http.server 8000
```
